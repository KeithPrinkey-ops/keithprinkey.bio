# Contributing to keithprinkey.bio

Thank you for considering contributing to my personal portfolio project! This site showcases projects, services, and technology integrations that reflect my current skill set and is built using Laravel, Laravel Livewire, TailwindCSS, Alpine.js, and more.

## 📌 Repository Requirements for GitHub Education Submission

This repository meets GitHub’s Education Pack selection criteria:

* ✅ It includes a detailed repository **description**.
* ✅ A `README.md` file is present to provide technical and contextual information.
* ✅ A valid `LICENSE` file defines what others can and cannot do with the source code.
* ✅ One or more **open issues** are available to work on.
* ✅ This `CONTRIBUTING.md` file outlines how to get involved.

---

## 🧩 Types of Contributions Welcome

* Fixing open issues (see the [Issues](../../issues) tab)
* Improving accessibility or responsiveness
* Refining Blade components or UI sections
* Adding light educational features or sample content (e.g., Code snippet highlights)
* Improving documentation (README, this file, etc.)

---

## 📜 Code of Conduct

By contributing, you agree to uphold a constructive and respectful tone. Harassment, trolling, or discrimination of any kind will not be tolerated.
